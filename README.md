# CICS Vault — NEU Document Repository

**Live Site:** https://neu-lab-project-prof-elec.vercel.app

A secure document repository for the College of Information and Computer Studies at New Era University.

## Features
- Google Sign-In restricted to @neu.edu.ph accounts
- Students select their program (CS, IT, IS, EMC) on first login
- Browse and download official CICS documents
- Admin panel to upload, manage documents, and monitor activity
- Activity logs with daily, weekly, monthly filters
